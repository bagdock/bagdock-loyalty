# Changelog

## 0.1.0 (2026-04-04)

### Features

- Initial public release extracted from the Bagdock monorepo.
- Typed HTTP client for the Loyalty API with retries and structured errors.
- **Members** — create, get, update, list members.
- **Links** — create and manage referral links.
- **Events** — track custom loyalty events.
- **Enrollments** — manage program enrollments.
- **Subscriptions** — manage recurring loyalty subscriptions.
- **Embed tokens** — create, validate, and list.
- **Domains** — manage custom loyalty domains.
- **Platform** — multi-operator support with `forOperator()`.
